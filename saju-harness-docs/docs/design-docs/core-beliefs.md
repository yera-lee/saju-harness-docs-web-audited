# Core Beliefs

## Belief 1: Saju is a frame for self-understanding

이 웹 서비스는 사주를 절대적 예언이 아니라 자기이해 프레임으로 사용한다.

## Belief 2: Love and relationship are the strongest entry points

초기 사용자는 복잡한 명리학보다 자신의 연애 성향, 끌리는 사람, 반복되는 관계 패턴에 더 빠르게 반응한다.

## Belief 3: Trust is more important than fear

공포를 자극하면 단기 전환은 가능할 수 있지만, 장기 신뢰와 리텐션을 해친다.

## Belief 4: Calculation and interpretation must be separated

계산 엔진과 해석 엔진이 섞이면 검증과 개선이 어려워진다.

## Belief 5: The first report should be short, personal, and shareable

첫 리포트는 깊이보다 즉각적인 공감과 다음 행동 유도가 중요하다.
