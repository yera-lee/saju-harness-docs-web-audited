# MVP Scope

## MVP Goal

사용자가 3분 이내에 첫 사주 기반 연애 성향 리포트를 받고, 궁합 분석으로 이어질 동기를 느끼게 한다.

## Included

- Landing page
- Onboarding
- Birth profile
- Basic chart_json
- First love pattern report
- Safety filtered interpretation
- Compatibility CTA

## Excluded

- Full compatibility report
- Payment
- Daily fortune
- Monthly flow
- Advanced astrology dashboard
- International birth support

## Success Metric Hypotheses

- onboarding completion rate
- report completion rate
- compatibility CTA click rate
- report save/share intent
- unsafe output rate
