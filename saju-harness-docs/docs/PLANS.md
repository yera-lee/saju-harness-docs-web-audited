# PLANS.md

## Roadmap

## Phase 0: Harness Setup

Goal:
- Agent들이 일할 수 있는 문서 체계를 구축한다.

Deliverables:
- AGENTS.md
- ARCHITECTURE.md
- PRODUCT_SENSE.md
- product specs
- quality score
- security/reliability docs
- initial exec plan

## Phase 1: MVP Onboarding + First Report

Goal:
- 사용자가 생년월일 정보를 입력하고 첫 연애 성향 리포트를 받는다.

Deliverables:
- Web landing page
- Responsive web onboarding flow
- Birth profile storage
- Basic chart calculation
- First love pattern report
- Compatibility CTA

## Phase 2: Compatibility Report

Goal:
- 두 사람의 정보를 기반으로 궁합 리포트를 제공한다.

Deliverables:
- Partner input flow
- Compatibility calculation
- Compatibility report UI
- Share CTA

## Phase 3: Retention Features

Goal:
- 사용자가 반복 방문할 이유를 만든다.

Deliverables:
- Daily flow
- Monthly relationship flow
- Saved report URLs
- Notification strategy

## Phase 4: Monetization

Goal:
- 무료 리포트에서 유료 심화 리포트로 확장한다.

Deliverables:
- Paid deep report
- Payment integration
- Report preview
- Purchase history
