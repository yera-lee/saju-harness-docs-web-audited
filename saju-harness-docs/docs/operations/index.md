## Purpose

이 디렉터리는 문서 기반 구현을 실제 운영 가능한 작업 흐름으로 바꾸기 위한 기준을 정의한다.

## Operating Principle

운영 기준은 다음 질문에 답해야 한다.

- 지금 무엇을 해야 하는가?
- 어디까지 하면 완료인가?
- 막히면 어디에 기록하는가?
- 다음 사람이 이어받을 수 있는가?
- 하네스 결과가 신뢰 가능한가?
