# DESIGN.md

## Design Direction

이 웹 서비스는 전통 점술 서비스처럼 무겁고 복잡한 인상을 주지 않는다.  
사주를 감각적인 자기이해 콘텐츠처럼 경험하게 만든다.

## Visual Principles

1. 차분함
   - 과도한 붉은색, 금색, 무속적 이미지는 피한다.
   - 신뢰감 있는 배경과 여백을 사용한다.

2. 개인화
   - 사용자의 닉네임과 선택한 관심사를 결과 화면에 반영한다.
   - “나를 위한 리포트”라는 느낌을 준다.

3. 읽기 쉬움
   - 긴 텍스트는 카드 단위로 나눈다.
   - 섹션별 제목을 명확히 제공한다.
   - 모바일 화면에서 한 번에 읽기 부담스럽지 않게 한다.

4. 부드러운 전환
   - 입력 → 분석 중 → 결과 → 궁합 CTA 흐름이 자연스러워야 한다.

## UX Principles

- 첫 입력은 3분 이내에 끝나야 한다.
- 출생시간을 몰라도 진행 가능해야 한다.
- 결과는 너무 길지 않게 시작하고, 심화 리포트로 확장한다.
- 궁합 CTA는 강매처럼 보이면 안 된다.

## Landing Page Direction

Hero message:
> 나는 어떤 연애를 반복할까?

Sub message:
> 생년월일을 바탕으로 나의 연애 성향, 끌리는 사람, 반복되는 관계 패턴을 분석해드려요.

Primary CTA:
> 내 연애 패턴 분석하기

Secondary CTA:
> 그 사람과의 궁합 보기
